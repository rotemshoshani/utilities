SETUP GUIDE: "session" CLI tool
================================

This is a guide for a Claude Code instance to set up the "session" tool on a
second machine. Follow every step exactly. Do not skip anything.

The "session" tool is an interactive CLI that lets the user select which
~/projects repos to push or pull using fzf, then runs Claude Code on each
selected project. It is used at the start and end of every work session to
keep two machines in sync.

Usage:
  session done   — end of work session, push selected projects to GitHub
  session sync   — start of work session, pull latest from GitHub


STEP 1: Install fzf (no sudo needed)
-------------------------------------
Run:
  git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf && ~/.fzf/install --bin

This puts the fzf binary at ~/.fzf/bin/fzf.


STEP 2: Create ~/bin directory
------------------------------
Run:
  mkdir -p ~/bin


STEP 3: Add ~/bin and ~/.fzf/bin to PATH in ~/.bashrc
------------------------------------------------------
Add these two lines to the END of ~/.bashrc (before any nvm or other tool
blocks if they exist). If either line already exists, skip it.

  [ -d "$HOME/bin" ] && export PATH="$HOME/bin:$PATH"
  [ -d "$HOME/.fzf/bin" ] && export PATH="$HOME/.fzf/bin:$PATH"

IMPORTANT: These must go in .bashrc, not just .profile, because terminal
emulators open non-login shells that only source .bashrc.


STEP 4: Create the script at ~/bin/session
-------------------------------------------
Create the file ~/bin/session with the EXACT content below, then run:
  chmod +x ~/bin/session

--- BEGIN ~/bin/session ---
#!/usr/bin/env bash
set -euo pipefail

PROJECTS_DIR="$HOME/projects"
FZF_CMD="${FZF_CMD:-fzf}"

# Try common fzf locations
if ! command -v "$FZF_CMD" &>/dev/null; then
    for candidate in "$HOME/.fzf/bin/fzf" "/usr/bin/fzf" "/usr/local/bin/fzf"; do
        if [ -x "$candidate" ]; then
            FZF_CMD="$candidate"
            break
        fi
    done
fi

# Colors
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
CYAN='\033[0;36m'
MAGENTA='\033[0;35m'
BLUE='\033[0;34m'
BOLD='\033[1m'
DIM='\033[2m'
NC='\033[0m'

usage() {
    echo -e "${BOLD}session${NC} — manage work sessions across machines"
    echo ""
    echo -e "  ${BOLD}session done${NC}    End of session: push selected projects"
    echo -e "  ${BOLD}session sync${NC}    Start of session: pull selected projects"
    echo ""
    echo -e "  ${DIM}Scans git repos in $PROJECTS_DIR${NC}"
    exit 0
}

print_legend() {
    echo -e "${BOLD}Status guide:${NC}"
    echo -e "  ${RED}3?${NC}  = new files git doesn't know about yet"
    echo -e "  ${YELLOW}2M${NC}  = files you edited but haven't saved to git"
    echo -e "  ${GREEN}1S${NC}  = files ready to be committed (staged)"
    echo -e "  ${CYAN}4↑${NC}  = commits saved locally but not yet on GitHub"
    echo -e "  ${MAGENTA}2↓${NC}  = commits on GitHub you don't have locally"
    echo -e "  ${RED}no remote${NC} = not connected to GitHub"
    echo -e "  ${DIM}clean${NC}     = fully synced, nothing to do"
    echo -e "  ${DIM}[main]${NC}    = the branch you're on"
    echo ""
}

check_deps() {
    if ! command -v "$FZF_CMD" &>/dev/null; then
        echo -e "${RED}fzf is required but not found.${NC}"
        echo "  Install: git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf && ~/.fzf/install --bin"
        exit 1
    fi
    if ! command -v claude &>/dev/null; then
        echo -e "${RED}claude CLI is required but not found.${NC}"
        exit 1
    fi
}

# List direct-child git repos in ~/projects
get_git_projects() {
    for dir in "$PROJECTS_DIR"/*/; do
        [ -d "$dir/.git" ] && basename "$dir"
    done | sort
}

# Return 0 if project has uncommitted changes or unpushed commits
has_changes() {
    local dir="$PROJECTS_DIR/$1"
    (
        cd "$dir" || return 1
        [ -n "$(git status --porcelain 2>/dev/null)" ] && return 0
        [ -n "$(git log --oneline '@{upstream}..HEAD' 2>/dev/null)" ] && return 0
        # No upstream at all but has commits
        if ! git rev-parse --abbrev-ref '@{upstream}' &>/dev/null; then
            [ -n "$(git log --oneline -1 2>/dev/null)" ] && return 0
        fi
        return 1
    )
}

# Return 0 if project is behind upstream
is_behind() {
    local dir="$PROJECTS_DIR/$1"
    (
        cd "$dir" || return 1
        [ -n "$(git log --oneline 'HEAD..@{upstream}' 2>/dev/null)" ] && return 0
        return 1
    )
}

# Format one line for fzf display (with ANSI colors)
format_line() {
    local name="$1"
    local dir="$PROJECTS_DIR/$name"

    local info
    info=$(
        cd "$dir" || return
        branch=$(git branch --show-current 2>/dev/null || echo "detached")
        untracked=$(git ls-files --others --exclude-standard 2>/dev/null | wc -l)
        modified=$(git diff --name-only 2>/dev/null | wc -l)
        staged=$(git diff --cached --name-only 2>/dev/null | wc -l)
        unpushed=$(git log --oneline '@{upstream}..HEAD' 2>/dev/null | wc -l)
        behind=$(git log --oneline 'HEAD..@{upstream}' 2>/dev/null | wc -l)
        has_upstream=1
        git rev-parse --abbrev-ref '@{upstream}' &>/dev/null || has_upstream=0
        echo "$branch $untracked $modified $staged $unpushed $behind $has_upstream"
    )

    local branch untracked modified staged unpushed behind has_upstream
    read -r branch untracked modified staged unpushed behind has_upstream <<< "$info"

    local parts=()
    [ "$untracked" -gt 0 ]  && parts+=("\033[0;31m${untracked}?\033[0m")
    [ "$modified" -gt 0 ]   && parts+=("\033[1;33m${modified}M\033[0m")
    [ "$staged" -gt 0 ]     && parts+=("\033[0;32m${staged}S\033[0m")
    [ "$unpushed" -gt 0 ]   && parts+=("\033[0;36m${unpushed}\xe2\x86\x91\033[0m")
    [ "$behind" -gt 0 ]     && parts+=("\033[0;35m${behind}\xe2\x86\x93\033[0m")
    [ "$has_upstream" -eq 0 ] && parts+=("\033[0;31mno remote\033[0m")

    local status_str
    if [ ${#parts[@]} -gt 0 ]; then
        status_str=$(IFS=' '; echo -e "${parts[*]}")
    else
        status_str=$(echo -e "\033[2mclean\033[0m")
    fi

    printf "%-28s \033[2m%-14s\033[0m %b" "$name" "[$branch]" "$status_str"
}

# Write the fzf preview script to a temp file and echo its path
make_preview_script() {
    local f
    f=$(mktemp /tmp/session-preview-XXXXX.sh)
    cat > "$f" << 'PREVIEW'
#!/usr/bin/env bash
name=$(echo "$1" | awk '{print $1}')
dir="$HOME/projects/$name"
cd "$dir" 2>/dev/null || { echo "Cannot access $dir"; exit 1; }

B='\033[1m'; D='\033[2m'; C='\033[0;36m'; Y='\033[1;33m'; G='\033[0;32m'; R='\033[0;31m'; N='\033[0m'

echo -e "${B}${name}${N}"
printf "${D}"; printf '─%.0s' {1..40}; printf "${N}\n\n"

branch=$(git branch --show-current 2>/dev/null || echo "detached")
echo -e "${D}Branch:${N}  $branch"

tracking=$(git rev-parse --abbrev-ref "@{upstream}" 2>/dev/null)
if [ -n "$tracking" ]; then
    echo -e "${D}Tracks:${N}  $tracking"
else
    echo -e "${D}Tracks:${N}  ${R}none${N}"
fi

echo -e "${D}Latest:${N}  $(git log --oneline -1 2>/dev/null || echo 'no commits')"
echo ""

unpushed=$(git log --oneline '@{upstream}..HEAD' 2>/dev/null)
if [ -n "$unpushed" ]; then
    c=$(echo "$unpushed" | wc -l)
    echo -e "${C}${B}Unpushed commits ($c):${N}"
    echo "$unpushed" | head -15
    [ "$c" -gt 15 ] && echo -e "${D}  ... and $((c-15)) more${N}"
    echo ""
fi

behind=$(git log --oneline 'HEAD..@{upstream}' 2>/dev/null)
if [ -n "$behind" ]; then
    c=$(echo "$behind" | wc -l)
    echo -e "${Y}${B}Behind upstream ($c):${N}"
    echo "$behind" | head -15
    [ "$c" -gt 15 ] && echo -e "${D}  ... and $((c-15)) more${N}"
    echo ""
fi

status=$(git status --short 2>/dev/null)
if [ -n "$status" ]; then
    c=$(echo "$status" | wc -l)
    echo -e "${G}${B}Working tree ($c):${N}"
    echo "$status" | head -25
    [ "$c" -gt 25 ] && echo -e "${D}  ... and $((c-25)) more${N}"
else
    echo -e "${D}Working tree clean${N}"
fi

echo ""
printf "${D}"; printf '─%.0s' {1..40}; printf "${N}\n"
echo -e "${D}?  = new file git doesn't track${N}"
echo -e "${D}M  = file you edited${N}"
echo -e "${D}S  = file staged (ready to commit)${N}"
echo -e "${D}↑  = commits not yet on GitHub${N}"
echo -e "${D}↓  = commits on GitHub you're missing${N}"
PREVIEW
    chmod +x "$f"
    echo "$f"
}

# Extract project names from fzf selection (one per line)
extract_names() {
    while IFS= read -r line; do
        echo "$line" | awk '{print $1}'
    done
}

# Run claude for each project with the given skill
run_claude_on() {
    local skill="$1"
    shift
    local names=("$@")

    local failed=()
    local succeeded=()

    for name in "${names[@]}"; do
        local dir="$PROJECTS_DIR/$name"
        echo -e "${BLUE}━━━ ${name} ━━━${NC}"
        if (cd "$dir" && claude -p "/$skill" --allowedTools Bash 2>&1); then
            succeeded+=("$name")
            echo -e "${GREEN}done${NC}"
        else
            failed+=("$name")
            echo -e "${RED}failed${NC}"
        fi
        echo ""
    done

    echo -e "${BOLD}━━━ Summary ━━━${NC}"
    [ ${#succeeded[@]} -gt 0 ] && echo -e "${GREEN}OK:     ${succeeded[*]}${NC}"
    [ ${#failed[@]} -gt 0 ]    && echo -e "${RED}Failed: ${failed[*]}${NC}"
}

# ── done ────────────────────────────────────────────────────────────
run_done() {
    echo -e "${BOLD}End of Session${NC}"
    echo ""

    local lines=()
    local total=0

    while IFS= read -r name; do
        total=$((total + 1))
        if has_changes "$name"; then
            lines+=("$(format_line "$name")")
        fi
    done < <(get_git_projects)

    if [ ${#lines[@]} -eq 0 ]; then
        echo -e "${DIM}All $total projects are clean. Nothing to push.${NC}"
        exit 0
    fi

    echo -e "${DIM}${#lines[@]}/$total projects have changes${NC}"
    echo ""
    print_legend
    echo -e "${DIM}TAB select | Ctrl-A all | ENTER confirm | ESC cancel${NC}"
    echo ""

    local preview
    preview=$(make_preview_script)

    local selected
    selected=$(
        printf '%b\n' "${lines[@]}" | "$FZF_CMD" \
            --multi \
            --ansi \
            --header="Select projects to push  |  ?=new  M=edited  S=staged  ↑=not pushed  ↓=not pulled" \
            --preview="bash '$preview' {}" \
            --preview-window=right:50%:wrap \
            --border=rounded \
            --prompt="done > " \
            --height=80% \
            --bind='ctrl-a:select-all,ctrl-d:deselect-all'
    ) || { rm -f "$preview"; echo "Cancelled."; exit 0; }

    rm -f "$preview"

    local names=()
    while IFS= read -r n; do
        names+=("$n")
    done < <(echo "$selected" | extract_names)

    echo ""
    echo -e "${BOLD}Pushing ${#names[@]} project(s)...${NC}"
    echo ""

    run_claude_on "0-done" "${names[@]}"

    echo ""
    echo -e "${BOLD}Session ended.${NC}"
}

# ── sync ────────────────────────────────────────────────────────────
run_sync() {
    echo -e "${BOLD}Start of Session${NC}"
    echo ""

    echo -e "${DIM}Fetching all remotes...${NC}"
    local pids=()
    for dir in "$PROJECTS_DIR"/*/; do
        if [ -d "$dir/.git" ]; then
            (cd "$dir" && git fetch --quiet 2>/dev/null) &
            pids+=($!)
        fi
    done
    for pid in "${pids[@]}"; do wait "$pid" 2>/dev/null || true; done
    echo -e "${DIM}Done.${NC}"
    echo ""

    local lines=()
    while IFS= read -r name; do
        lines+=("$(format_line "$name")")
    done < <(get_git_projects)

    if [ ${#lines[@]} -eq 0 ]; then
        echo "No git projects found in $PROJECTS_DIR"
        exit 0
    fi

    print_legend
    echo -e "${DIM}TAB select | Ctrl-A all | ENTER confirm | ESC cancel${NC}"
    echo ""

    local preview
    preview=$(make_preview_script)

    local selected
    selected=$(
        printf '%b\n' "${lines[@]}" | "$FZF_CMD" \
            --multi \
            --ansi \
            --header="Select projects to sync  |  ?=new  M=edited  S=staged  ↑=not pushed  ↓=not pulled" \
            --preview="bash '$preview' {}" \
            --preview-window=right:50%:wrap \
            --border=rounded \
            --prompt="sync > " \
            --height=80% \
            --bind='ctrl-a:select-all,ctrl-d:deselect-all'
    ) || { rm -f "$preview"; echo "Cancelled."; exit 0; }

    rm -f "$preview"

    local names=()
    while IFS= read -r n; do
        names+=("$n")
    done < <(echo "$selected" | extract_names)

    echo ""
    echo -e "${BOLD}Syncing ${#names[@]} project(s)...${NC}"
    echo ""

    run_claude_on "0-sync" "${names[@]}"

    echo ""
    echo -e "${BOLD}Ready to work.${NC}"
}

# ── main ────────────────────────────────────────────────────────────
check_deps
[ $# -lt 1 ] && usage

case "$1" in
    done) run_done ;;
    sync) run_sync ;;
    -h|--help|help) usage ;;
    *) echo -e "${RED}Unknown command: $1${NC}"; echo ""; usage ;;
esac
--- END ~/bin/session ---


STEP 5: Verify it works
------------------------
Open a NEW terminal (so .bashrc reloads), then run:
  session help

You should see the usage info. Then test the interactive selector:
  session sync

This will fetch all remotes, show the project list with fzf, and let you
pick which ones to pull.


STEP 6: Prerequisites reminder
-------------------------------
These must already be in place on the machine (they probably are):
  - git configured with user.name and user.email
  - GitHub SSH keys set up (git push/pull should work without password prompts)
  - Claude Code CLI installed and authenticated
  - ~/projects directory containing the same git repos as the other machine


TROUBLESHOOTING
---------------
Problem: "session: command not found"
  -> Open a new terminal, or run: source ~/.bashrc
  -> Verify: which session  (should print ~/bin/session)

Problem: "fzf is required but not found"
  -> Run: git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf && ~/.fzf/install --bin
  -> Then add to .bashrc: [ -d "$HOME/.fzf/bin" ] && export PATH="$HOME/.fzf/bin:$PATH"

Problem: Claude fails on a project with permission errors
  -> The script runs: claude -p "/0-done" --allowedTools Bash
  -> If that's not enough, the user's Claude Code permission settings may need
     to allow Bash tool execution. Check ~/.claude/settings.json.

Problem: A project shows "no remote"
  -> That project has no GitHub upstream. Set one with:
     cd ~/projects/PROJECTNAME && git remote add origin git@github.com:USER/REPO.git
